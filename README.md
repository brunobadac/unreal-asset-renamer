# Unreal Asset Renamer
A simple pipeline automation tool written in Python to rename Unreal Engine assets using naming conventions.

## Features
- Automatically adds Unreal naming prefixes
- Helps maintain consistent asset naming
- Useful for Unreal Engine and pipeline workflows

## Supported Prefixes
| Asset Type | Prefix |
|------------|--------|
| Static Mesh | SM_ |
| Texture | T_ |
| Material | M_ |
| Blueprint | BP_ |
| Animation | AN_ |

## How to use
Run the script:
```
python main.py
```

Enter the folder path and asset type, and the script will rename the files automatically.

## Why this tool
This tool was created as a simple pipeline utility to help maintain consistent naming conventions for Unreal Engine assets.
